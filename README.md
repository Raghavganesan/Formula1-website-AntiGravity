# F1 Experience Website

An interactive F1-inspired website showcasing the pinnacle of motorsport.

## Features
- **Interactive Intro**: "Get to Know the Sport" modal with a smooth entrance.
- **Championship Data**: View Drivers and Constructors standings.
- **Dynamic Content**: Real-time data loading (simulated) with clean UI.
- **Responsive Design**: Optimized for various screen sizes.
